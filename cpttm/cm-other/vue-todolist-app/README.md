# vue-todolist-app

一個使用 Vue.js 製作的待辦清單應用程式。

## 特色

- 新增、刪除、標記完成待辦事項
- 資料即時更新，介面簡潔易用

## 安裝與使用

1. 下載專案：
   ```bash
   git clone https://github.com/BiuwuLOK/vue-todolist-app.git
   cd vue-todolist-app
   ```

2. 安裝依賴：
   ```bash
   npm install
   ```

3. 啟動開發伺服器：
   ```bash
   npm run serve
   ```

4. 打開瀏覽器並造訪 [http://localhost:8080](http://localhost:8080)
