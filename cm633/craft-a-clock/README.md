# cm633-craft-a-clock

一個用於 CM633 課程的「Craft a Clock」專案，展示如何使用 HTML、CSS 與 JavaScript 製作互動式時鐘網頁應用。

## 特色

- 純前端技術（HTML/CSS/JavaScript）實作
- 動態顯示目前時間
- 可自訂時鐘外觀（如顏色、樣式等）
- 響應式設計，適用於不同裝置

## 專案結構

```
/
├── index.html      # 主要網頁檔案
├── style.css       # 樣式表
├── script.js       # 時鐘邏輯與互動
└── 其他資源
```

## 安裝與執行

1. 下載或 clone 此專案至本地端：

   ```bash
   git clone https://github.com/BiuwuLOK/cm633-craft-a-clock.git
   ```

2. 於瀏覽器中打開 `index.html`，即可看到時鐘運作。

## 使用說明

- 開啟網頁後，即可看到目前時間的時鐘
- 可依需求修改 `style.css` 或 `script.js` 來自訂時鐘外觀與功能
